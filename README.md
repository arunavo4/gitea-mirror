# GiteaMirror

**GiteaMirror** is an open-source tool to automatically mirror repositories from GitHub to your self-hosted Gitea instance. Designed for developers, teams, and organizations who want to retain full control of their code while still collaborating on GitHub.

- 🔁 Sync public, private, or starred GitHub repos to Gitea
- 🕹️ Web UI to manage and trigger mirror jobs
- 🧠 Smart filtering and job queue with logs
- 🛠️ Works with personal access tokens (GitHub + Gitea)
- 🐳 Fully Dockerized + can be self-hosted in minutes
